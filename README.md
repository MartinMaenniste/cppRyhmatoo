# Kirjeldus
See grupitöö on kloon teadatuntud mängust *Minesweeper*, kasutades C++ programmeerimiskeelt loogikaks ning SFML teeki graafika jaoks.

# Funktsionaalsused
 - Saab valida mängulaua suuruse (kõrguse ja laiuse eraldi)
 - Saab valida, mitu protsenti ruutudest on pommid

# Projekti ülesehitus
|_Pildid
|__*.png - Kõik pildifailid mängulaua ruutude jaoks.
|_ main.cpp - Loob mängu objekti ja alustab mängutsüklit.
|_Minesweeper.cpp
|_Minesweeper.h - Mängu loogika.
